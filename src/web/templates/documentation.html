Документация API

Анализатор можно использовать с помощью запросов к API.

  1. Для загрузки файла необходимо отправить POST запрос по ссылке http://0.0.0.0/upload.
  2. Для запуска обработки файла отправляется POST запрос по ссылке http://0.0.0.0/process/<имя файла>, в параметре data необходимо указать названия модулей обработки текста:
    • ner - Извлечение имен ученых;
    • topic - Определение тематики;
    • rb - Определение метрик ридабилити;
    • term - Извлечение терминов).
    Названия перечисляются в одно строковое значение (например, 'topic,term').
  3. В ответе на этот запрос возвращается список task_id в формате json.
  4. Для каждого task_id происходит выполнение задания по обработке текста. По ссылке http://0.0.0.0/status/ отображаются данные о статусе выполнения задания. После получения статуса "SUCCESS" по этому же запросу возвращается имя обработанного файла в виде "processed/.txt" (например, "processed/70d2a667-8910-4e98-835d-1545b67caab8.txt").
  5. По ссылке http://0.0.0.0/files/<имя обработанного файла> отображается результат выполнения задания.
